# テストシナリオテンプレート

このファイルは、テスト実施時にコピーして使用してください。
各項目を実施したらチェックボックスにチェックを入れてください。

## 1. フロー管理機能（リスト画面）

### 1.1. フローの新規作成
- [ ] フロー一覧画面で「新規フロー作成」ボタンをタップする。
- [ ] 新しいフローが一覧に追加され、編集画面に遷移することを確認する。
- [ ] デフォルトのフロー名が表示されていることを確認する。

### 1.2. フロー名の変更
- [ ] フロー一覧画面で既存のフローを長押し（または編集アイコンをタップ）し、名前変更オプションを選択する。
- [ ] 新しいフロー名を入力し、保存する。
- [ ] フロー一覧画面でフロー名が更新されていることを確認する。

### 1.3. フローの削除
- [ ] フロー一覧画面で既存のフローを長押し（または削除アイコンをタップ）し、削除オプションを選択する。
- [ ] 確認ダイアログが表示されることを確認する。
- [ ] 削除を確定し、フローが一覧から削除されることを確認する。

### 1.4. フロー一覧の表示
- [ ] アプリ起動時にフロー一覧画面が正しく表示されることを確認する。
- [ ] 複数のフローが作成されている場合、すべてが一覧に表示されることを確認する。
- [ ] スクロール操作がスムーズに行えることを確認する。

### 1.5. フローのエクスポート
- [ ] フロー一覧画面でエクスポートしたいフローを選択する。
- [ ] エクスポートメニューから「Zip形式」を選択する。
- [ ] Zipファイルが生成され、共有メニューが表示されることを確認する。
- [ ] エクスポートメニューから「Canvas形式(JSON)」を選択する。
- [ ] Obsidianを選択し、Canvasファイルの保存が完了することを確認する。

### 1.6. フローの複数選択・一括削除
- [ ] フロー一覧画面で複数選択モードに入る。
- [ ] 複数のフローを選択する。
- [ ] 「一括削除」ボタンをタップする。
- [ ] 確認ダイアログが表示されることを確認する。
- [ ] 削除を確定し、選択したフローがすべて一覧から削除されることを確認する。

### 1.7. フローのソート
- [ ] フロー一覧画面でソートオプションを開く。
- [ ] 「作成日時順」「更新日時順」「名前順」などのソート基準を選択する。
- [ ] 選択した基準に従ってフローの表示順が変更されることを確認する。
- [ ] 昇順/降順の切り替えが正しく機能することを確認する。

### 1.8. フローの検索
- [ ] フロー一覧画面で検索バーをタップする。
- [ ] フロー名の一部を入力する。
- [ ] 入力した文字列を含むフローのみが一覧に表示されることを確認する。
- [ ] 検索結果がない場合に、その旨が正しく表示されることを確認する。
- [ ] 検索バーをクリアすると、すべてのフローが再度表示されることを確認する。

## 2. カード操作機能（エディタ画面）

### 2.1. カードの新規作成
- [ ] フロー編集画面で「新規カード作成」ボタン（または画面の空いている場所をタップ）をタップする。
- [ ] 新しいカードが画面に追加されることを確認する。
- [ ] デフォルトのカードタイトルが表示されていることを確認する。

### 2.2. カードのコンテンツ編集
- [ ] カードをタップして編集モードに入る。
- [ ] タイトルと説明を入力し、保存する。
- [ ] カードに更新されたタイトルと説明が表示されることを確認する。

### 2.3. カードの添付ファイル（URL）
- [ ] カード編集画面で「URLを追加」を選択する。
- [ ] 有効なURLを入力し、保存する。
- [ ] カードにリンクアイコンとURLが表示されることを確認する。
- [ ] リンクをタップしてブラウザが起動することを確認する。
- [ ] 添付したURLを削除できることを確認する。

### 2.4. カードの添付ファイル（画像）
- [ ] カード編集画面で「画像を追加」を選択する。
- [ ] カメラまたはギャラリーから画像を選択する。
- [ ] 画像のサムネイルがカードに表示されることを確認する。
- [ ] サムネイルをタップして画像が全画面表示されることを確認する。
- [ ] 添付した画像を削除できることを確認する。

### 2.5. カードの添付ファイル（動画）
- [ ] カード編集画面で「動画を追加」を選択する。
- [ ] ギャラリーから動画を選択する。
- [ ] 動画のサムネイルがカードに表示されることを確認する。
- [ ] サムネイルをタップして動画が再生されることを確認する。
- [ ] 添付した動画を削除できることを確認する。

### 2.6. カードの添付ファイル（その他ファイル）
- [ ] カード編集画面で「ファイルを追加」を選択する。
- [ ] ドキュメントピッカーからファイル（PDF, テキストファイルなど）を選択する。
- [ ] ファイル名とアイコンがカードに表示されることを確認する。
- [ ] タップして対応するアプリでファイルが開かれることを確認する。
- [ ] 添付したファイルを削除できることを確認する。

### 2.7. カードの移動
- [ ] カードを長押ししてドラッグし、画面上の別の位置に移動する。
- [ ] カードがスムーズに移動し、指を離した位置に固定されることを確認する。

### 2.8. カードのスタイル変更
- [ ] カードを選択し、スタイル編集メニューを開く。
- [ ] カードの背景色を変更し、即座に反映されることを確認する。
- [ ] カードのサイズ（幅・高さ）を変更し、即座に反映されることを確認する。
- [ ] 変更したスタイルが保存され、再表示後も維持されることを確認する。

### 2.9. カードの削除
- [ ] カードを長押し（または削除アイコンをタップ）し、削除オプションを選択する。
- [ ] 確認ダイアログが表示されることを確認する。
- [ ] 削除を確定し、カードが画面から削除されることを確認する。

## 3. 矢印（関連）操作機能

### 3.1. 単方向矢印の作成
- [ ] 始点となるカードをタップし、次に関連付けたい終点のカードをタップする。
- [ ] 始点から終点に向かう単方向の矢印が正しく描画されることを確認する。

### 3.2. 双方向矢印への変更
- [ ] 既存の単方向矢印がある状態で、作成時と逆の順番（終点→始点）でカードを再度タップする。
- [ ] 矢印が双方向になることを確認する。

### 3.3. 矢印の削除
- [ ] 既存の矢印をタップする。
- [ ] 矢印が画面から削除されることを確認する。

## 4. UI操作機能（エディタ画面）

### 4.1. ズームイン/アウト
- [ ] フロー編集画面でピンチイン/ピンチアウト操作を行う。
- [ ] 画面がスムーズに拡大/縮小されることを確認する。
- [ ] 拡大/縮小後もカードや矢印が正しく表示されることを確認する。

### 4.2. パン（画面移動）
- [ ] フロー編集画面でドラッグ操作を行う。
- [ ] 画面がスムーズに移動することを確認する。
- [ ] 移動後もカードや矢印が正しく表示されることを確認する。

### 4.3. 整列機能
- [ ] 複数のカードを選択する。
- [ ] 整列メニューから「左揃え」「中央揃え」「右揃え」などを選択する。
- [ ] 選択したカードが指定通りに整列されることを確認する。
- [ ] 「等間隔に配置」などを選択し、カードが均等に配置されることを確認する。

### 4.4. セクション操作（階層移動）
- [ ] 任意のカードをダブルタップする。
- [ ] ダブルタップしたカードの内部に移動し、そのカードが親セクションとして表示されることを確認する。
- [ ] 移動したセクション内部で、新しいカードを作成したり、既存のカードを編集したりできることを確認する。
- [ ] 画面下部などに表示される「セクションアップ」FABボタンをタップする。
- [ ] 親セクション（元のフロー）に戻り、子セクション（ダブルタップしたカード）が元の位置に表示されていることを確認する。
- [ ] 複数の階層を移動した場合でも、正しく親セクションに戻れることを確認する。

### 4.5. シースルー（See-Through）機能
- [ ] カードが重なっている状態で、前面のカードをシースルー表示に切り替える。
- [ ] 前面のカードが半透明になり、背面のカードが透けて見えることを確認する。
- [ ] シースルー状態でもカードの操作（移動、タップなど）が可能であることを確認する。
- [ ] シースルー表示を元に戻せることを確認する。

## 5. データ永続化

### 5.1. アプリ再起動後のデータ保持
- [ ] 複数のフローとカード、矢印、スタイル変更を作成する。
- [ ] アプリを完全に終了し、再度起動する。
- [ ] 作成したフロー、カード、矢印、スタイルがすべて元の状態で表示されることを確認する。

## 6. 国際化対応

### 6.1. 言語切り替え
- [ ] アプリの設定画面（またはOSの設定）で言語を日本語から英語（またはその逆）に切り替える。
- [ ] アプリ内のUIテキスト（ボタン、メニュー、カードのデフォルトタイトルなど）が切り替わった言語で表示されることを確認する。
- [ ] 言語切り替え後も機能が正常に動作することを確認する。

## 7. その他

### 7.1. データベースの初期化
- [ ] 設定画面などから「データベース初期化」ボタンをタップする。
- [ ] データの完全削除に関する警告ダイアログが表示されることを確認する。
- [ ] ダイアログで「キャンセル」をタップし、データが削除されていないことを確認する。
- [ ] 再度初期化ボタンをタップし、ダイアログで「初期化」を確定する。
- [ ] すべてのフロー、カード、矢印などのデータが削除され、アプリが初期状態に戻ることを確認する。
- [ ] アプリを再起動し、データが空の状態が維持されていることを確認する。
